# Caixa Eletrônico no Terminal (Java)

Simulação de um caixa eletrônico em Java executado no terminal.

## Funcionalidades
- Consultar saldo
- Depositar valores
- Sacar valores com validação de saldo
- Menu interativo no console

## O que foi praticado
- Estruturas de decisão (if / switch)
- Estruturas de repetição (while / do-while)
- Validação de entradas
- Controle de fluxo

## Observações
Projeto de estudo. Versão inicial, com planos de evolução futura aplicando métodos e melhorias de organização.
